# Playwright Java Automation Framework

## About
End-to-end test automation framework 
for Sauce Demo e-commerce application
built using Playwright Java.

## Tools Used
- Playwright Java 1.42.0
- TestNG 7.8.0
- Maven
- Java 17

## Framework Structure
- base — Browser setup and teardown
- page — Page Object Model classes
- test — Test classes
- utils — Config reader utility

## Test Coverage
- Login with valid credentials
- Add multiple items to cart
- Complete checkout flow
- Order confirmation verification

## How to Run
1. Clone repository
2. Run: mvn test
3. View reports in test-output folder
